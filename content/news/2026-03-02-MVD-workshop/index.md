# Outline

The MaterialVital-Digital project aims to support the digitalization of processes involved in the development of Engineered Living Materials (ELMs). To achieve this, experimental and development processes will be semantically described using ontologies, while experimental work will be documented in electronic laboratory notebooks (ELNs).

Galaxy will be used to integrate ELM-related tools and to design reproducible workflows as well as training material. Integration between ELNs and Galaxy will facilitate the transfer of experimental data, for example between Galaxy and ELN platforms such as RSpace. In addition, CKAN will be used for storing and managing generated datasets.

# Workshop aim

During this two-day workshop, the INM team presented several user stories related to ELM development. These examples helped the project partners better understand the requirements and opportunities for digitalizing the development process. The presented use cases focused on:

* Biofabrication via electrospinning
* Ink characterization using rheology
* Genetic engineering of genetically modified microorganisms (GMMs)

Following these presentations, project partners provided updates on key components of the digital infrastructure, including semantic concepts, Galaxy workflows, CKAN data management, PMDco, RSpace integration, and ontology development.

This workshop represents the first step toward developing a comprehensive strategy for the digitalization of material development, covering the full pipeline from experimental processes to computational analysis.
