# EuroScienceGateway News

This page contains announcements of interest to the EuroScienceGateway Project.

## Contribute

To add a news item, create a new folder within the news folder using the year bucket layout, for example `content/news/2026/my-news-item/index.md`. Make sure to add `esg` to subsites, or the news item won't show up here!
