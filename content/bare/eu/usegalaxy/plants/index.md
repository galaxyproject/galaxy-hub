# Welcome to Galaxy for Plant Biology

The __Plant Analysis Workbench__ is a comprehensive set of analysis tools and consolidated workflows. The workbench is based on the [Galaxy framework](https://galaxyproject.org),
which guarantees simple access, easy extension, flexible adaption to personal and security needs, and sophisticated analyses independent of command-line knowledge.

# Get started

Are you new to Galaxy, or returning after a long time, and looking for help to get started? Take a [__guided tour__](https://plants.usegalaxy.eu/tours/core.galaxy_ui) through Galaxy's user interface.

# Partners
