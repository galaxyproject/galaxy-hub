From 2019 to 2022, EOSC-Nordic project was creating and developping Galaxy Climate and from 2018, French Museum of Natural History was creating and developping Galaxy Ecology intiative. From these 2 communities, some tools were created so users can use of produce Essential Ocean Variables related data products.

# Use Galaxy for operationalizing EOV

## E0Vs access

[Copernicus Marine Environment Monitoring Service CMEMS](https://ecology.usegalaxy.eu/?tool_id=toolshed.g2.bx.psu.edu/repos/ecology/copernicusmarine/copernicusmarine/2.3.0+galaxy0) Galaxy tool allows users to retrieve data from Copernicus marine data store.

## EOVs creation

Essential Ocean Variables production is done by labeled services. Users can't produce it ontheir own. That said, some Galaxy tools and workflows can be used to produce datasets containing variables similar than those proposed as official EOVs.
